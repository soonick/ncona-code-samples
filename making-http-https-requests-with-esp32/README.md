# Introduction to ESP32 WiFi Manager

Before you running the examples, replace `<YOUR_SSID>` and `<YOUR_PASSWORD>` with your SSID and password in `minimal-https/main/main.cpp` and `production-https/main/main.cpp`.

To build and upload the minimal example to ESP32 and start serial monitor:

```
make upload serial
```

To build and upload the production example to ESP32 and start serial monitor:

```
make upload-prod serial-prod
```
